#树莓派RaspberryPi  
  哥买的是B型，装的是2013-02-09-wheezy-raspbian，小派的镜像貌似挺难下的，用win32diskimage写的SD卡  
  连续运行几周后，cpu温度在55摄氏度左右（加装铜散热片）  
  
  默认用户是pi 密码为raspberry,root账户默认没有密码，但账户锁定。

  重新开启root账号，可由pi用户登录后，在命令行下执行

  sudo passwd root  
  
  执行此命令后系统会提示输入两遍的root密码，输入你想设的密码即可
